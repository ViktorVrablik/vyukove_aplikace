Nápověda

Ovládání

  • Přidat blok: klik nebo přetažení z panelu vlevo
  • Přesun: levé tlačítko + tažení
  • Editovat text: dvojklik na blok (mimo Start/Stop)
  • Pan: pravé tlačítko + tažení | Zoom: kolečko myši
  • Výběr více: táhněte rámeček nebo Shift+klik
  • Ctrl+C / Ctrl+V / Ctrl+Z / Del

Syntaxe

Proměnné: x = 5 · x += 1 · x++
Výstup: print(x) · print("text: " + x)
Vstup: x = read()
Pole: a = array[10] · a[i] = 5 · n = a.length
Podmínka: x > 5 · x == 0 · x >= 3 && x < 10
Více příkazů: x = 1; y = 2; print(x+y)
